## Installing Zig

NorthstarDB requires Zig 0.13.0 or later.

### Linux (Debian/Ubuntu)
```bash
# Download Zig
wget https://ziglang.org/download/0.13.0/zig-linux-x86_64-0.13.0.tar.xz
tar -xf zig-linux-x86_64-0.13.0.tar.xz

# Add to PATH
export PATH=$PATH:$(pwd)/zig-linux-x86_64-0.13.0
```

### macOS
```bash
brew install zig
```

### Windows
Download from [ziglang.org](https://ziglang.org/download/) and add to PATH.

## Building NorthstarDB

```bash
# Clone the repository
git clone https://github.com/nikopol/northstardb.git
cd northstardb

# Build
zig build

# Run tests
zig build test

# Run benchmarks
zig build run -- run --repeats 5
```

## Build Artifacts

- `zig-out/bin/bench` - Benchmark harness
- `zig-out/bin/northstardb` - Database library

## CI Requirements

The project uses continuous integration with:

- Regression gates: -5% throughput, +10% p99 latency
- Hardening tests nightly
- Baseline comparisons for performance validation
