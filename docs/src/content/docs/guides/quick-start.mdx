## Prerequisites

- **Zig** 0.13.0 or later
- Git

## Installation

```bash
# Clone the repository
git clone https://github.com/nikopol/northstardb.git
cd northstardb

# Build the project
zig build

# Run benchmarks
zig build run -- run

# Run tests
zig build test
```

## Basic Usage

```zig
const std = @import("std");
const db = @import("northstardb");

pub fn main() !void {
    var gpa = std.heap.GeneralPurposeAllocator(.{}){};
    defer _ = gpa.deinit();
    const allocator = gpa.allocator();

    // Open or create database
    var database = try db.Db.open(allocator, "mydb.northstar");
    defer database.deinit();

    // Start a write transaction
    var write_txn = try database.beginTxn(.{ .mode = .write });
    defer write_txn.commit() catch unreachable;

    // Write some data
    try write_txn.put("hello", "world");

    // Read data in a read transaction
    var read_txn = try database.beginTxn(.{ .mode = .read });
    defer read_txn.commit() catch unreachable;

    const value = try read_txn.get("hello");
    std.debug.print("Got: {s}\n", .{value});
}
```
