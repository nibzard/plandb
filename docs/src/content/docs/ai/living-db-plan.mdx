## Overview

This document describes the plan to transform NorthstarDB into a **"Living Database"** with autonomous intelligence.

Full details: [PLAN-LIVING-DB.md](https://github.com/nikopol/northstardb/blob/main/PLAN-LIVING-DB.md)

## Vision

A database that:
- **Understands** its own content
- **Optimizes** itself continuously
- **Communicates** in natural language
- **Learns** from its workload
